## 🚀 Live Demo
Once GitHub Pages finishes building (~1 minute), open the Pages URL from your repo's **Settings → Pages** section.

## 📱 Install as a PWA
1. Open the live URL in Chrome / Safari on your phone
2. Tap the browser menu → **Add to Home Screen**
3. The app installs like a native app 🎉

## 🗂 Structure
```
index.html          → app shell
styles.css          → dark, mobile-first UI
app.js              → all logic (uses localStorage)
manifest.json       → PWA metadata
service-worker.js   → offline cache
icon.svg            → app icon
```

## 🛠 Local Development
Just open `index.html` in a browser — no build step needed. For full PWA testing, serve it over HTTPS (e.g. `npx serve`).
